## 🌟 Overview

**MachAPI** is a powerful platform that allows users to **instantly create, deploy, and manage AI-powered APIs** without writing backend logic.

Instead of manually building endpoints, handling infra, and wiring LLMs — **MachAPI automates everything**.

## ⚙️ How It Works

1️⃣ User submits:

* 🔑 API key
* 🧠 Prompt
* 📥 Input schema
* 📤 Output format

2️⃣ MachAPI:

* 🔨 Stores configuration
* 🧠 Generates endpoint
* 🚢 Deploys backend logic

3️⃣ User gets:

```bash
POST https://ai-machapi.vercel.app/{username}/{custom-endpoint}
```

4️⃣ Send request → Get AI response 🎯

## 📥 Installation

### 1️⃣ Clone the Repo

```bash
git clone https://github.com/MihirRajeshPanchal/MachAPI
cd MachAPI
```

---

### 2️⃣ Install Dependencies

#### Frontend (Next.js)

```bash
cd frontend
npm install
```

#### Backend (FastAPI - Vercel)

```bash
cd backend
pip install -r requirements.txt
```

---

### 3️⃣ Run Locally

#### Frontend

```bash
npm run dev
```

#### Backend

```bash
uvicorn backend.app:app --reload
```
